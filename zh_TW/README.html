關於 Goravel ​

Goravel 是一個功能完備、具有良好擴展能力的 Web 應用程式框架。 作为一个起始脚手架帮助 Golang 开发者快速构建自己的应用。 作為起始腳手架幫助 Gopher 快速構建自己的應用。

框架設計與 Laravel 一致，簡化 PHP 開發者的學習曲線。 向 Laravel 致敬!

我們開放接受 stars、PRs 和 issues!

主要功能 ​

配置HTTP用戶認證用戶授權資料庫 ORM
資料庫遷移日誌快取GrpcArtisan 命令行
任務排程隊列事件系統文件存儲郵件
表單驗證MockHashCryptCarbon
擴展包開發測試本地化Session
