# Goravel Document

This is goravel official document.

## Contributing

We welcome contributions to the Goravel documentation. Please see the [contributing guide](en/prologue/contributions.md) for more information.

## Install

```
pnpm install
```

## Run dev

```
pnpm docs:dev
```

## Run Build

```
pnpm docs:build
```

## Deploy

We use Github Actions to deploy the document to the server.

Production: https://github.com/goravel/docs/actions/workflows/deploy.yml

Preview: https://github.com/goravel/docs/actions/workflows/deploy-pre.yml
